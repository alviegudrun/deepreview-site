如何获取 Paddle Vendor ID

1

访问 Paddle 商家后台 并登录您的账户

2

点击右上角的商家名称，选择「Account settings」

3

在「Account Info」选项卡中找到您的「Vendor ID」

4

将下面的代码中的 YOUR_PADDLE_VENDOR_ID 替换为您的实际 Vendor ID

配置代码示例

将以下代码中的 YOUR_PADDLE_VENDOR_ID 替换为您从Paddle后台获取的实际Vendor ID：

// Paddle 配置 const PADDLE_CONFIG = { vendorId: 'YOUR_PADDLE_VENDOR_ID', // ← 替换为您的实际Vendor ID environment: 'sandbox', // 测试完成后改为 'production' products: { yearly: 'pri_01k3a39jpkydhhwra9ne1s797m', monthly: 'pri_01k3a38rw99kgp3g64n2accx14' } }; // Paddle 客户端令牌 (测试环境) const PADDLE_CLIENT_TOKEN = 'test_4b05d1a77a742bd77ffa74e3c30';
注意： 在测试环境中使用 sandbox 模式，上线时请将 environment 改为 'production'， 并使用以 'live_' 开线的生产环境令牌。
